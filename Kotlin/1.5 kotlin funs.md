## What Is a Function?

A function groups reusable logic. It can take parameters and return a result.

```kotlin
val pi = 3.14

fun circumference(radius: Double) = 2 * pi * radius

val small = circumference(5.2)
val medium = circumference(6.7)
val large = circumference(10.0)
```

---

## Function Parameters

Functions can accept multiple parameters.

```kotlin
fun speed(distance: Double, time: Double) = distance / time

val avg = speed(321.8, 4.15)
```

### Named Arguments

Arguments can be passed by name.

```kotlin
val avg = speed(time = 4.15, distance = 321.8)
```

This improves readability and allows reordering.

---

## Default Arguments

Parameters can have default values.

```kotlin
fun speed(distance: Double, time: Double = 2.0) = distance / time

val walk = speed(10.2)
val fly  = speed(1368.7, 1.5)
```

---

## Function Body Types

### Expression Body

Used for simple logic.

```kotlin
fun circumference(radius: Double) = 2 * pi * radius
```

### Block Body

Used when multiple statements are required.

```kotlin
var counter = 0

fun increment() {
    counter += 1
}
```

Functions without a return value return `Unit`.

---

## Summary

- Functions make code reusable and readable
- Support multiple, named, and default parameters
- Expression bodies are concise
- Block bodies support complex logic
