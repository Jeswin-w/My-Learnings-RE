
# Keywords and Operators (Kotlin)

## Hard keywords

### as, as? – Type casting

```kotlin
val rawInput: Any = "user-1234"
val userId: String = rawInput as String
val safeUserId: String? = rawInput as? String
```

### break, continue – Loop control

```kotlin
for (i in 1..5) {
    if (i == 3) continue
    if (i == 5) break
    println(i)
}
```

### class, interface, object

```kotlin
interface Engine {
    fun start()
}

class Car : Engine {
    override fun start() {
        println("Car started")
    }
}

object Singleton {
    val version = 1
}
```

### if, else, when

```kotlin
val x = 10
val result = if (x > 5) "Big" else "Small"

val grade = when (x) {
    in 1..5 -> "Low"
    in 6..10 -> "High"
    else -> "Unknown"
}
```

### for, while, do

```kotlin
var i = 0

while (i < 3) {
    println(i)
    i++
}

do {
    println("Runs at least once")
} while (false)

for (item in listOf("A", "B", "C")) {
    println(item)
}
```

### is, !is – Type checks

```kotlin
fun printType(value: Any) {
    if (value is String) {
        println("String of length ${value.length}")
    }
}
```

### try, catch, finally, throw

```kotlin
try {
    val n = "abc".toInt()
} catch (e: NumberFormatException) {
    println("Invalid number")
} finally {
    println("Done")
}
```

## Soft keywords

### by – Delegation

```kotlin
interface Logger {
    fun log(msg: String)
}

class ConsoleLogger : Logger {
    override fun log(msg: String) = println(msg)
}

class Service(logger: Logger) : Logger by logger
```

### constructor, init

```kotlin
class User constructor(val name: String) {
    init {
        println("User created: $name")
    }
}
```

## Modifier keywords

### data, enum, sealed

```kotlin
data class User(val id: Int, val name: String)

enum class Status {
    ACTIVE, INACTIVE
}

sealed class Result {
    data class Success(val data: String) : Result()
    data class Error(val message: String) : Result()
}
```

### open, override

```kotlin
open class Base {
    open fun run() {}
}

class Derived : Base() {
    override fun run() {
        println("Running")
    }
}
```

### inline, reified

```kotlin
inline fun <reified T> printType() {
    println(T::class)
}
```

## Operators

### Equality

```kotlin
val x = "hello"
val y = "hello"

println(x == y)
println(x === y)
```

### Null safety

```kotlin
val name: String? = null
println(name?.length)
println(name ?: "Unknown")
```

### Ranges

```kotlin
for (i in 1..5) println(i)
for (i in 1..<5) println(i)
```

### Lambdas

```kotlin
val sum: (Int, Int) -> Int = { a, b -> a + b }
println(sum(2, 3))
```
